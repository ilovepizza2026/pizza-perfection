# Pizza Perfection 🍕

A modern React-based pizza ordering platform built with TypeScript and Vite. Browse our delicious pizza menu, explore premium toppings, and rate your favorites.

## ✨ Features

- **Interactive Pizza Menu**: Browse our curated selection of artisan pizzas with detailed descriptions and pricing
- **Smart Toppings Search**: Search and filter through our extensive toppings catalog by name or category
- **Star Rating System**: Rate pizzas with our intuitive 5-star rating component
- **Responsive Design**: Clean, mobile-friendly interface built with modern CSS
- **Type-Safe**: Fully typed with TypeScript for better developer experience

## 🛠️ Technologies Used

- **React 19** - Modern UI library with latest features
- **TypeScript 5.7** - Type-safe JavaScript development
- **Vite 6** - Fast build tool and development server
- **Modern CSS** - Responsive design with system fonts

## 🚀 Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd pizza-perfection
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and visit `http://localhost:5173`

## 📜 Available Scripts

- `npm run dev` - Start the development server with hot reloading
- `npm run build` - Build the production-ready application
- `npm run preview` - Preview the production build locally
